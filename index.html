Courser: an elective exploring project


Courser started as a learning platform for me to get used to the MEAN stack and learn a bit about the DOM tree. It's grown quite a bit from an idea I had to a small functioning website. The website uses MongoDB's built in search in order to find school courses that relate to the search terms. This part of the site actually works fairly well, although I've wanted to change the way it searches so that it's more accurate. I'm soon planning on making a blog for this site so I can track my progress. This is the biggest project I've worked on so far. I'm proud that I've created it all on my own and I hope that when I finish it, people will enjoy using it as much as I will.
